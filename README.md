# 🎮 Pokédex

Este projeto é uma Pokédex interativa desenvolvida em HTML, CSS e JavaScript puro, que consome a [PokéAPI](https://pokeapi.co/) para exibir informações sobre diferentes Pokémon. O objetivo é praticar conceitos de manipulação de DOM, consumo de APIs e estilização responsiva.

## 📝 Funcionalidades

- **Busca por nome ou número:** Digite o nome ou número do Pokémon para visualizar suas informações.
- **Navegação:** Use os botões "Prev" e "Next" para navegar entre os Pokémon.
- **Exibição de informações:** Mostra o nome, número e sprite animado do Pokémon.
- **Design responsivo:** Interface adaptável para diferentes tamanhos de tela.

## 🗂️ Estrutura de Pastas

```
index.html
css/
  style.css
js/
  script.js
Imagens/
favicons/
```

- **index.html:** Página principal do projeto.
- **css/style.css:** Estilos personalizados para a Pokédex.
- **js/script.js:** Lógica de busca, navegação e integração com a PokéAPI.
- **Imagens/:** Imagens utilizadas na interface (ex: imagem da Pokédex).
- **favicons/:** Ícones para diferentes dispositivos e navegadores.

## Como funciona:

1. **Busca:**  
   O usuário pode digitar o nome ou número do Pokémon no campo de busca e pressionar Enter. O sistema faz uma requisição à PokéAPI e exibe as informações do Pokémon correspondente.

2. **Navegação:**  
   Os botões "Prev" e "Next" permitem navegar sequencialmente entre os Pokémon.

3. **Exibição:**  
   O nome, número e sprite animado do Pokémon são exibidos na tela. Se o Pokémon não for encontrado, nada é exibido.

## Como rodar o projeto

1. Clone este repositório:

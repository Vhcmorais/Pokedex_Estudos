# 🎮 Pokédex - HTML, CSS and JavaScript

This project is an interactive Pokédex developed using pure HTML, CSS, and JavaScript. It consumes the PokéAPI to display information about different Pokémon. The main goal is to practice DOM manipulation, API consumption, and responsive web design concepts.

## 📝 Features

- **Search by name or number:** Enter a Pokémon's name or number to view its information.
- **Navigation:** Use the "Prev" and "Next" buttons to browse through Pokémon.
- **Information display:** Displays the Pokémon's name, number, and animated sprite.
- **Responsive design:** Interface adapts to different screen sizes.

## 🗂️ Folder Structure

```text
index.html
css/
  style.css
js/
  script.js
Imagens/
favicons/
```

- **index.html:** Main project page.
- **css/style.css:** Custom styles for the Pokédex.
- **js/script.js:** Search, navigation, and PokéAPI integration logic.
- **Imagens/:** Images used in the interface (e.g., Pokédex image).
- **favicons/:** Icons for different devices and browsers.

## How It Works

### 1. Search

The user can enter a Pokémon's name or number in the search field and press Enter. The application sends a request to the PokéAPI and displays information about the corresponding Pokémon.

### 2. Navigation

The "Prev" and "Next" buttons allow users to navigate sequentially through Pokémon.

### 3. Display

The Pokémon's name, number, and animated sprite are displayed on the screen. If the Pokémon is not found, no information is shown.

## Running the Project

1. Clone this repository:

```sh
git clone https://github.com/your-username/Pokedex_Estudos.git
```

2. Open the `index.html` file in your browser.

> No dependencies or local servers are required, as the project uses only pure HTML, CSS, and JavaScript.
